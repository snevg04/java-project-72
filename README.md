# Анализатор страниц
Данное приложение позволяет проверять различные ресурсы на SEO-пригодность. Ресурсы добавляются в базу данных для дальнейшей проверки, история проверок сохраняется. При проверке приложение совершает HTTP-запрос по указанному адресу, а из полученного ответа сохраняет код ответа, заголовок, название сайта и описание.
